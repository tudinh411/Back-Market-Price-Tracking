# Scraper 
* Developed a web application dedicated to tracking prices and uncovering the latest deals on Back Market.
* Leveraged Python for efficient website scraping, ensuring up-to-date product information.



  THIS APPLICATION:
-> Enables users to input their desired product names seamlessly.
-> Presents the product list in ascending order of prices for easy comparison.
-> Includes product rates and URL links, providing a comprehensive overview.
